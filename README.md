# Unique T-shirts Backend

Express + TypeScript + Mongoose server for Unique T-shirts.

## Environment Variables (.env)

- PORT=8080
- MONGODB_URI=mongodb+srv://...
- FRONTEND_ORIGIN=http://localhost:3000

## Scripts

- npm run dev — local development with nodemon
- npm run build — compile TypeScript
- npm start — run compiled server

## Render Deployment

Use the provided render.yaml for Blueprint deploy, or create a Web Service pointing to this repo.
